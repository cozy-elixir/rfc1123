# RFC2822

An implementation of RFC 2822.

## Installation

Add `:rfc2822` to the list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:rfc2822, <requirement>}
  ]
end
```

## Usage

For more information, see the [documentation](https://hexdocs.pm/rfc2822).
